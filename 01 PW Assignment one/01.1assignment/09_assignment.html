HTML Text Formatting

  1. u(underline)
    • This tag is used for underline
    • The u tag has been deprecated in HTML5
  2. b(bold)
    • This tag used for bold the element.
    • The b tag has been deprecated in HTML5
  3. i(italic)
    • This tag is used for emphasis words.
    • The i tag has been deprecated in HTML5
  4. a(ancher)
    • This tag is used for insert hyperlink to interlink pages.
    • The a tag has been deprecated in HTML5
